Contribution

JORDI OÑA

On the first Delivery, I implemented the external librarys: Assimp, MathGeoLib, DevIL and made them load geometry and textures with an structure based on models and base geometries.

For the Second Delivery, I implemented the base of the Scene structure: based on GameObjects and Components. I also added a quadtree to optimize static gameobjects searches. Skyboxes added too. I implemented Guizmos to edit transformations on GameObjects and the Game Clock, that enables PLAY/PAUSE/PLAYFRAME Modes. Mouse picking was another of my implementations.

In the last Delivery, I was literally four days searching information about how to embed Mono into your application. Once implemented in our engine, I implemented the method to link c# functions to c++ functions and created some of these functions. I prepared the 3 scripts to run the scene with the tank: "CameraController", "Tank", "myTurret". I also implemented the structure code to pass functions from Mono to your app and link them to the current script.
